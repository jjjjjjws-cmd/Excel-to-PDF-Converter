## 💻 사용 방법

### 간단 모드

**언제 사용:**
- 1개 열만 처리
- 간단한 작업
- 빠른 설정

**사용법:**
1. 프로그램 실행
2. "간단 모드" 선택
3. 엑셀 파일 선택
4. 링크 열 입력 (예: H)
5. 시트 선택 (전체 선택 가능)
6. 저장 폴더 선택
7. 변환 시작

**처리 방식:**
- 선택한 시트를 순차적으로 처리
- 크롬 1개 사용
- 모든 PDF를 같은 폴더에 저장

---

### 고급 모드

**언제 사용:**
- 여러 열 동시 처리
- 대량 작업
- 빠른 완료 필요

**사용법:**
1. 프로그램 실행
2. "고급 모드" 선택
3. 엑셀 파일 선택
4. 처리할 열 추가 (H, I, J... [+] 버튼)
5. 시트 선택 (체크박스)
6. 각 (시트×열) 조합마다 폴더 선택
   - 예: 시트1-H → C:\1_H\
   - 예: 시트1-I → C:\1_I\
   - 예: 시트2-H → C:\2_H\
7. 변환 시작

**처리 방식:**
- 모든 작업 동시 처리 (병렬)
- 크롬 여러 개 사용
- 각 작업마다 다른 폴더에 저장

**예시:**
```
3개 시트 × 2개 열 = 6개 작업 동시 실행

시트1-H열 → C:\7월_H\    ┐
시트1-I열 → C:\7월_I\    │
시트2-H열 → C:\8월_H\    ├─ 동시 실행!
시트2-I열 → C:\8월_I\    │
시트3-H열 → C:\9월_H\    │
시트3-I열 → C:\9월_I\    ┘

시간: 약 6배 빠름!
```

## ❓ FAQ

### Q1. 간단 모드와 고급 모드 차이는?
**A.** 간단 모드는 1개 열만 순차 처리, 고급 모드는 여러 열을 동시 처리합니다.

### Q2. 고급 모드가 느린 컴퓨터에서도 작동하나요?
**A.** 작동은 하지만 많은 메모리를 사용합니다. 4GB RAM 이하라면 간단 모드 권장.

### Q3. 작업 중 다른 크롬을 사용할 수 있나요?
**A.** 네! v3.2는 독립적인 크롬을 사용하므로 다른 크롬 창에 영향 없습니다.

### Q4. 고급 모드에서 최대 몇 개까지 동시 실행 가능한가요?
**A.** 제한은 없지만 컴퓨터 성능에 따라 다릅니다. 보통 10~20개 권장.

### Q5. 파일명은 어떻게 저장되나요?
**A.** 웹페이지 제목으로 저장됩니다. 중복 시 _1, _2 등이 추가됩니다.

### Q6. 모드 전환하면 설정이 초기화되나요?
**A.** 네, 안전을 위해 초기화됩니다.
